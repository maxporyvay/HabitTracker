Данный треккер поможет вам запоминать задачи, поставленные на день, а так же вести контроль за привычками.

С помощью функции <Plans for the day> вы можете:
	Просмотреть запланированный на сегодня задачи: <Show plan in this day>;
	Добавить новую задачу: <Add plan in this day>, для этого в строке для ввода напечатайте текст новой задачи и нажмите <Add>;
	Выйти в главное меню: <To menu>.

С помощью функции <Fast adding plans> вы можете добавить задачу на любой нужный день, для этого введите дату по соответствующим полям и текст новой задачи, после чего нажмите <Add>.

С помощью функции <Shedule> вы можете:
	Просмотреть статистику по своим привычкам в определенном месяце: <Saved months> --> <Habit tracker> --> <Выбор нужного месяца> --> <Choose>
	Посмотреть планы на конкретный день: <Saved months> --> <Plans> --> <Выбор нужного месяца> --> <Choose> --> <Введите нужный день> --> <Enter>
	Добавить новый месяц в список используемых месяцев: <Add month>

С помощью функции <Exit> вы выйдете из приложения.
